# The Engine Room Heist — PF2e Encounter Report

An independent simulation-backed balance analysis for **The Engine Room Heist**, evaluated for a party of four level-4 Pathfinder Second Edition characters.

The published report is available at:

https://dballesteros7.github.io/engine-room-heist-report/

The dependency-free Python Monte Carlo model, assumptions, publication seed,
tests, and full output are in [`simulation/`](simulation/README.md).

```sh
python3 simulation/engine_room_mc.py
```
